# Secondary Review Batch

## Purpose

A compact package for a stronger secondary reviewer to inspect multiple
high-escalation tasks or one complex task with compressed context.

Note: Batch review is opt-in. The default is per-task secondary review. Use this
template only when the primary reviewer explicitly groups related high-risk
tasks.

## Questions For Secondary Reviewer

- What subtle issues might the primary reviewer have missed?
- Which tasks look safe to approve?
- Which tasks should be revised or blocked?

## Output Expectation

The secondary reviewer should produce one review note per task, or one batch
summary plus task-specific verdicts.
